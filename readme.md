#Ejercicio 17.1

##### Contexto

Escribir un algoritmo que indique al usuario si el número ingresado es positivo o negativo.
Caso de Prueba 1
Input
num = 2
Output
"Positivo"
Caso de Prueba 2
Input
num = -3
Output
"Negativo"

##### Pseudocodigo

1. Inicio
2. Ingrese num
3.  ¿ num es mayor que 0?
 -> si ->Imprimir "el numero es positivo"
 	no->Imprimir "el numero es negativo"
4. Fin
